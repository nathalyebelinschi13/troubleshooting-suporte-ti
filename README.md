# Troubleshooting de Suporte Técnico

## Problema: Usuário não consegue acessar o sistema
**Causa possível:** senha incorreta ou usuário bloqueado  
**Solução:** redefinição de senha e desbloqueio

## Problema: Sistema lento
**Causa possível:** alta carga no servidor ou internet do usuário  
**Solução:** validar conexão e status do sistema

## Problema: Erro ao carregar página
**Causa possível:** cache do navegador  
**Solução:** limpar cache e tentar novamente
